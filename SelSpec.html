CSS Selector Function/Specificity
Class Specificity 1: Overriden by any other selector
Used to apply CSS elements in very broad cases
Tag Specificity 2: Overriden by everything but Class
Used to apply CSS elements over a specific set of broad cases
ID Specificity 3: Overrides all selectors but !Important
Used to apply CSS elements to specific items
!Important Specificity 1: Overrides any other selector, cannot be overriden
Used to apply CSS elements that should NEVER be overriden, only used in special cases
